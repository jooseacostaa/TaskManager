# 📝 Task Manager (Java Console App)

A simple console-based task manager built in Java.  
This project was developed as a beginner exercise to practice object-oriented programming, collections, and user
interaction via the terminal.

---

## 🚀 Features

- Add new tasks
- View all tasks
- Mark tasks as completed
- Delete tasks
- Interactive menu system

---

## 🧠 Concepts Used

- Object-Oriented Programming (OOP)
- Classes and objects
- ArrayList (Java Collections)
- Control flow (loops, conditionals, switch)
- User input with Scanner
- Method decomposition

---

## 📂 Project Structure

src/

- Main.java → Program entry point and menu handling
- GestorTareas.java → Task management logic
- Tarea.java → Task model

---

## ▶️ How to Run

Clone the repository:
git clone https://github.com/your-username/task-manager-java.git

Navigate to the project folder:
cd task-manager-java

Compile the project:
javac *.java

Run the application:
java Main
